Have Questions? Look Here.

Is MyBookfolio 100% Free?

Yes, currently MyBookfolio is 100% free, both instructors and cadets can benfit from the tools that MyBookfolio provides.

How Can I register a battalion?

1. In order to register a battalion one instructor from the battalion needs to register that battalion at: https://www.mybookfolio.com/auth/register.
2. Once one instructor as registered the battalion and created his account all other instructors can join the battalion using the battalion code.
3. A battalion code is a unique identifer that both instructors and cadets can use to join a battalion.

For help or more information, please contact us at: support@mybookfolio.com

How can cadets register into a battalion?

Cadets can register into a battalion by creating a account, when creating a account cadets will be asked for a Battalion Code, a Battalion Code is a unqiue identifer that links the cadet to the battalion. Instructors must share the battalion code with cadets since its the only way to pair a battalion and cadet.

Is there a cadet limit per battalion?

Currently there is no limit of cadets per battalion.
